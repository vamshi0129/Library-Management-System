# Library Management System
This is a java (swing) based application that manages book , users , transactions efficiently.


---

## Features

- Add ,remove ,update books
- User registration
- Search book (by title , by author)
- Issue and return book  
- Fine calculation  


---
## Setup

-place all files in same file names java
-Compile all by  javac .*java
-Run the UI  java LibraryGUI
